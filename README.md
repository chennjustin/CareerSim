# CareerSim

AI 模擬面試平台，透過多輪對話與智慧分析，幫助您更有自信地回答每一個問題！

## 🎯 專案概述

CareerSim 是一個基於 React + Vite + Tailwind CSS 建構的 AI 面試模擬平台。平台提供真實的面試場景模擬、多輪對話練習和詳細的回饋報告。

## ✨ 核心功能

- **📅 日曆檢視** - 以日曆形式管理所有面試活動，支援月檢視和列表檢視
- **💬 模擬面試** - 多輪對話式面試體驗，支援不同的 AI 面試官性格（友善/正式/壓力測試）
- **📊 智慧報告** - 詳細的評分分析和改進建議，包括表達、內容、結構和語言四個維度
- **📋 活動管理** - 完整的面試活動列表，支援篩選和狀態管理

## 🛠️ 技術棧

- **前端框架**: React 18 + TypeScript
- **建構工具**: Vite
- **樣式**: Tailwind CSS
- **路由**: React Router v6
- **日期處理**: date-fns
- **圖示**: Lucide React
- **資料儲存**: localStorage (偽後端)

## 🚀 快速開始

### 安裝相依套件

```bash
npm install
```

### 啟動開發伺服器

```bash
npm run dev
```

專案將在 `http://localhost:3000` 啟動

### 建構生產版本

```bash
npm run build
```

### 預覽生產建構

```bash
npm run preview
```

## 🎨 設計特色

- **極簡風格** - 受 Apple 和 ChatGPT 啟發的簡潔設計
- **專業配色** - 基於自訂調色板的專業配色方案
- **流暢動畫** - 平滑的過渡效果和互動動畫
- **響應式設計** - 適配各種螢幕尺寸

## 📝 使用說明

1. **建立面試活動** - 在 Dashboard 頁面點擊「新增活動」按鈕
2. **開始面試** - 點擊活動卡片進入面試房間
3. **回答問題** - 與 AI 面試官進行多輪對話（通常 5 輪）
4. **檢視報告** - 面試結束後自動產生詳細的分析報告
5. **改進練習** - 根據報告建議進行針對性練習
